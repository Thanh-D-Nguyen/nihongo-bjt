# Learning Experience Digest

Default tier: balanced. Escalate to deep-reasoning when a feature changes motivation loops, competition, habit mechanics, or learner wellbeing.

## Must read

- `docs/spec/index.md`
- `docs/spec/compact/06_learner_ui_modules.md`
- `docs/spec/compact/11_learning_effectiveness_experience.md`

## Conditional reads

- Sharing/privacy: `compact/07_security_privacy.md`
- Monetized limits/ads: `compact/08_monetization.md`
- Media/provenance/schema/API: `compact/02_database_prisma.md`, `compact/03_backend_api_registry.md`
- Testing/release gates: `compact/10_testing_acceptance.md`

## Done

- Feature supports focus, comprehension, habit, remediation, or visible progress.
- Audio/video/animation is user-controlled and accessible.
- Progress uses real persisted events/rollups.
- Competition and sharing are opt-in and privacy-safe.
- UI avoids shame, anxiety, clutter, autoplay, and distraction loops.

## Avoid

- Autoplay media in study/exam flows.
- Fake progress, fake ranks, fake opponents, fake coaching.
- Infinite-scroll social feeds in core learning.
- Pay-to-win or public shaming.
- Color-only meaning or inaccessible media.

## Check commands

- Run available lint/typecheck/test/build scripts.
- Smoke mobile and desktop states.
- Check i18n, reduced-motion/quiet-mode behavior where implemented.
- Verify media provenance/license fields for external assets.

## Escalate

Escalate for exam integrity, learner wellbeing, social pressure, leaderboard design, monetized competition, or privacy-sensitive sharing.
