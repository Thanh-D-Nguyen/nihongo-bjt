# Japan-Vietnam Localization Digest

Default tier: balanced. Escalate to review-security for legal/privacy wording and deep-reasoning for product tone conflicts.

## Mission

Make Vietnamese and Japanese product language natural, professional, learner-safe, and culturally appropriate.

## Must optimize

- natural Vietnamese UI
- professional Japanese UI
- consistent English fallback
- clear error messages
- useful empty states
- supportive coaching copy
- admin labels
- legal/compliance wording

## Must reject

- mixed machine-like English/Vietnamese labels
- awkward terms like untranslated feature jargon when a natural term exists
- shame-based copy
- vague errors
- legal claims without human review
- hard-coded user-facing strings
