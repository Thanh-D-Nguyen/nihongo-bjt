# BJT Assessment Psychometrics Digest

Default tier: deep-reasoning for exam blueprint/scoring; balanced for item copy review.

## Mission

Make BJT quizzes and mock exams credible, calibrated, and useful for remediation.

## Must optimize

- mock exam blueprint
- section balance: listening, listening-reading, reading
- skill tags
- difficulty index
- distractor quality
- question discrimination proxy
- time pressure analysis
- estimated score/band clarity
- remediation mapping

## Must reject

- question without correct answer
- weak or implausible distractors
- untagged items
- score presented as official BJT result
- repeated exposed questions
- quiz result without explanation/remediation
- analytics based on fake attempts

## Must read

- `docs/spec/compact/03_backend_api_registry.md`
- `docs/spec/compact/06_learner_ui_modules.md`
- `docs/spec/compact/10_testing_acceptance.md`
- `company/gates/assessment-quality-gate.md`

## Done

- every item has skill, difficulty, answer, explanation, remediation
- scoring says estimated where applicable
- timing is tracked
- item quality risks are documented
