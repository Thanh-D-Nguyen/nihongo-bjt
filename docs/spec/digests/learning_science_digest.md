# BJT Learning Science Digest

Default tier: balanced. Escalate to deep-reasoning for learning algorithms, motivation loops, exam feedback, or learner wellbeing.

## Mission

Make the product produce real BJT learning progress, not just engagement.

## Must optimize

- cognitive load
- spacing
- retrieval practice
- interleaving
- feedback quality
- motivation
- habit loop
- mistake recovery
- confidence calibration
- comeback after absence

## Must reject

- too many choices at once
- shame-based copy
- meaningless XP
- fake progress
- quiz without remediation
- flashcard without context
- streak punishment
- autoplay distraction
- social pressure during focused study

## Review targets

- learner dashboard
- flashcard review
- quiz result
- learning path
- analytics/coaching
- comeback mode
- onboarding/placement
- coaching messages

## Must read

- `docs/spec/compact/06_learner_ui_modules.md`
- `docs/spec/compact/11_learning_effectiveness_experience.md`
- `company/gates/learning-quality-gate.md`

## Done

- learner knows what to do next
- mistakes map to remediation
- progress is real and persisted
- UX supports focus and confidence
- accessibility/comfort options are respected
