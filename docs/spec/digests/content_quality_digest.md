# BJT Content Quality Digest

Default tier: balanced. Escalate to deep-reasoning for canonical content taxonomy or high-risk language decisions.

## Mission

Keep Japanese learning content accurate, natural, business-relevant, and level-appropriate.

## Must optimize

- dictionary sense quality
- natural business Japanese examples
- keigo/teineigo/sonkeigo/kenjougo correctness
- grammar explanation context
- kanji/vocab level fit
- duplicate detection
- content quality score
- remediation content links

## Must reject

- machine-like examples
- duplicate low-value entries
- grammar explanations without context
- level mismatch
- unsupported translations
- media without provenance
- unreviewed imported canonical content

## Must read

- `docs/spec/compact/02_database_prisma.md`
- `docs/spec/compact/03_backend_api_registry.md`
- `docs/spec/compact/06_learner_ui_modules.md`
- `docs/spec/compact/11_learning_effectiveness_experience.md`

## Done

- content has source/provenance
- examples sound natural and useful
- level and tags are justified
- quality issues are reviewable in admin
