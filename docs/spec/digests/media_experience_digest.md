# BJT Media Experience Digest

Default tier: balanced. Escalate to code-heavy for media pipelines and review-security for uploads/external fetch.

## Mission

Use audio, image, color, motion, video, and postcards to support learning without distracting from Japanese content.

## Must optimize

- pronunciation and listening comprehension
- visual context and memory cues
- readable Japanese text
- accessible controls
- captions/transcripts
- provenance/license traceability
- quiet/reduced-motion modes
- media fallback states

## Must reject

- autoplay in study/quiz/exam
- decorative clutter around learning content
- inaccessible color contrast
- color-only meaning
- media without provenance/license metadata
- unsafe uploads or external fetch
- video/audio with no learning objective

## Review targets

- daily-life hub media
- listening content
- pronunciation audio
- video lessons
- media library
- generated postcards
- share images/OG assets

## Must read

- `docs/spec/compact/07_security_privacy.md`
- `docs/spec/compact/11_learning_effectiveness_experience.md`
- `company/gates/media-quality-gate.md`

## Done

- media is user-controlled
- alt/captions/transcripts exist where applicable
- license/provenance is stored
- contrast and motion are accessible
- privacy-safe fallback exists
