# BJT Content and Assessment Rubrics

## Purpose

Make NihonGo BJT learning quality measurable, reviewable, and production-ready.

## Content Quality Rubric

Review Japanese learning content for:

- natural business Japanese usage
- correct keigo/teineigo/sonkeigo/kenjougo
- Vietnamese explanation clarity
- Japanese/Vietnamese terminology consistency
- level fit for BJT and learner stage
- example sentence usefulness in workplace context
- duplicate or low-quality content
- media provenance/license metadata when media is attached
- reading-assist compatibility for Japanese text
- no misleading financial, legal, medical, or gambling advice

## Assessment Rubric

Review quiz/mock exam items for:

- skill tag
- difficulty tag
- correct answer existence
- plausible distractors
- explanation for correct and incorrect choices
- remediation link to lesson/flashcard/reading practice
- time pressure rules when applicable
- section balance for BJT mock exam
- estimated score/band wording that does not claim official score
- no obvious item reuse that leaks the answer

## Learning Outcome Rubric

Review flows for:

- clear next best study action
- manageable cognitive load
- retrieval practice
- spaced review
- mistake recovery
- confidence calibration
- progress tied to mastery, not only activity count
- no shame-based copy
- comeback mode for returning learners

## Blockers

- quiz item without correct answer or explanation
- BJT score presented as official
- content that teaches unnatural or wrong Japanese
- financial/gambling topic presented as advice instead of language/practical context
- user weakness shown publicly without opt-in
