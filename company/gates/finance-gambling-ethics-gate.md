# Finance and Gambling Ethics Gate

Use for housing, tax, insurance, pension, banking, remittance, lottery, stocks, crypto, FX, investment, and other high-risk life-in-Japan contexts.

## Pass checklist

- Content is framed as Japanese/BJT learning, not advice.
- Risk category is identified.
- Disclaimer is shown where needed.
- Source/provenance/date is stored for current or external information.
- No buy/sell/hold recommendation exists.
- Lottery content teaches vocabulary/probability and does not encourage purchase.
- No fake portfolio, fake winnings, fake odds, or fake urgency.
- No affiliate/ad placement interrupts core learning.
- User can hide/mute sensitive topics where applicable.
- Personalized scenarios do not infer wealth, holdings, gambling behavior, immigration status, or tax status.
- Remediation links teach language/comprehension skills.
- Admin publish action is permissioned and audited.

## Evidence

- content path
- source/provenance fields
- disclaimer/i18n keys
- UI screenshot or route path
- tests/checks or documented gap
- reviewer and owner

## Blockers

- financial recommendation
- gambling encouragement
- missing disclaimer for high-risk topic
- unverified current facts
- dark-pattern monetization
- privacy-sensitive inference
