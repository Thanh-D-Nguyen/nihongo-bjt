# Assessment Quality Gate

Use for BJT quiz, mock exam, scoring, item banks, explanations, and remediation.

## Pass checklist

- Each question has skill tag.
- Each question has difficulty.
- Each question has exactly one correct answer unless explicitly multi-answer.
- Each question has explanation.
- Each question has remediation links.
- Distractors are plausible and diagnostically useful.
- Score/band is clearly marked estimated where applicable.
- Time pressure is recorded.
- Repeated/exposed items are controlled.
- Listening, listening-reading, and reading balance is documented for mock exams.
- Item analytics can flag low-accuracy or high-confusion questions.

## Evidence

- schema/API path
- item sample or validation rule
- scoring test
- analytics/event path
- known gaps and owner

## Blockers

- missing correct answer
- official-score wording for estimated score
- fake attempts/results
- no remediation from wrong answers
- untagged production item bank
