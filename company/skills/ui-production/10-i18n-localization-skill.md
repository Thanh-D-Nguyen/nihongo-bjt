# I18n and Localization Skill

## When to Use

Use for any user-facing label, message, error, empty state, admin action, learner coaching message, legal/privacy copy, or Japanese/Vietnamese UI.

## Required Checks

- Text uses i18n keys.
- Vietnamese is natural and professional.
- Japanese is polite and context-appropriate.
- English fallback is coherent.
- Error messages are actionable.
- Estimated score/progress wording is not misleading.
- No accidental mixed language unless intentional.

## Anti-Patterns

- Hard-coded production labels.
- Machine-like Vietnamese.
- Raw technical keys in UI.
- Vague success messages.
- Shame-based learner copy.
- Legal/compliance text edited without review.

## Output Checklist

- i18n keys listed
- locale files updated
- tone risks checked
- raw-label risk checked
