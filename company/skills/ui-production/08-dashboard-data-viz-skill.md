# Dashboard Data Visualization Skill

## When to Use

Use for dashboards, analytics, monetization reports, assessment reports, learning progress, admin operations metrics, and content quality metrics.

## Required Checks

- Labels are human-readable.
- Date range and data freshness are visible.
- Empty and degraded states exist.
- Legend/tooltips exist where needed.
- Estimates are clearly labeled.
- Precision is not misleading.
- Metrics are backed by real events/rollups/provider contracts.

## Anti-Patterns

- Fake trend delta.
- Fake growth percentage.
- Raw event names in chart UI.
- Unlabeled estimated BJT score.
- Decorative charts without decision value.
- Comparing periods without explaining date range.

## Output Checklist

- metric source named
- date range/freshness named
- estimate vs official status named
- empty/degraded behavior named
