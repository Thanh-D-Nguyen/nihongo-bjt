# Sensory Media and Motion Skill

## When to Use

Use for audio, image, video, postcard, animation, color, sound, haptics, illustrations, and visual feedback.

## Required Checks

- Each media element has a learning purpose.
- Audio/video is user-started, not autoplay.
- Captions/transcripts exist where relevant.
- Motion supports state change, not decoration.
- Reduced motion is respected.
- Image/video/postcard has alt text and provenance where needed.
- Color does not carry meaning alone.
- Sensory elements can be muted, paused, or skipped.

## Good Uses

- Listening practice controls.
- Pronunciation replay.
- Context image for daily life vocabulary.
- Postcard only after explicit user share action.
- Microinteraction for answer saved, card reviewed, or retry queued.

## Anti-Patterns

- Background music during study by default.
- Confetti for weak mastery.
- Video competing with Japanese reading.
- Motion loops that draw attention away from the task.
- Postcards exposing private score/user data.

## Output Checklist

- media purpose named
- controls/fallbacks named
- accessibility/provenance checked
- distraction risk checked
