# Learning Focus and Cognitive Load Skill

## When to Use

Use for dashboard, daily hub, flashcards, learning paths, comeback, remediation, progress, and study session pages.

## Required Checks

- One primary study action is obvious.
- Secondary actions are grouped or deferred.
- The page avoids more than 3 competing decisions above the fold.
- The learner can recover from mistakes with a concrete plan.
- Long Japanese text is chunked and supported.
- Review and next-step recommendations are backed by real data.
- Breaks, comeback, and low-energy states are supported without shame.

## Learning UI Patterns

- Today panel: what to do now, why it matters, expected time.
- Mistake panel: wrong answer, reason, skill tag, remediation link.
- Review queue: due count, estimated time, priority.
- Confidence calibration: estimated, not official, never exaggerated.
- Comeback mode: smaller task, visible win, no guilt copy.

## Anti-Patterns

- Too many cards and no priority.
- XP/streaks as the main progress signal.
- Praise that hides poor mastery.
- Failure copy that blames the learner.
- Endless scroll during focused study.
- Ads/social prompts inside active study.

## Output Checklist

- primary action named
- cognitive-load risk addressed
- remediation path named
- progress data source named
