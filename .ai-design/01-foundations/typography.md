# Typography

## Font strategy
Use system font stack unless project already defines a branded font.

## Scale
- Page title: 28–36px desktop, 22–28px mobile.
- Section title: 18–24px.
- Card title: 16–18px.
- Body: 14–16px.
- Metadata: 12–13px.

## Japanese readability
Japanese text must have:
- comfortable line height: 1.6–1.8 for long reading.
- no cramped kanji-heavy blocks.
- furigana or reading support where product requires it.
- clear distinction between Japanese, Vietnamese, English, and romaji.

## Rules
- Do not overuse bold.
- Use weight hierarchy before adding colors.
- Dictionary headwords may be larger and bolder.
- Explanations should scan easily with short paragraphs.
