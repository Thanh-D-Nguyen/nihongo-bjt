# Spacing

## Base grid
Use 4px base spacing.

## Common values
- 4: micro gap
- 8: compact inner gap
- 12: form/control spacing
- 16: card padding mobile
- 20–24: card padding desktop
- 32–48: section spacing

## Rules
- Search and dictionary pages may be denser.
- Learning screens need more breathing room.
- Admin dashboards may use compact density but must not feel cramped.
- Mobile pages must avoid horizontal overflow.
