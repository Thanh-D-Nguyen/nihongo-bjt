# Layout Grid

## Desktop
Use split layouts for productivity:
- Search/list left column: 360–460px.
- Detail panel: flexible.
- Sticky search/filter area when useful.

## Tablet
Prefer two-column where space allows; collapse gracefully.

## Mobile
Use single-column flow.
Detail pages may become bottom sheets or full-screen sheets.

## Rules
- Primary action must remain reachable.
- Do not hide important learning context behind hover-only interactions.
- Use max-width for reading content.
