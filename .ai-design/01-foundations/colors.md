# Colors

## Brand palette
- `brand.navy`: primary serious action color.
- `brand.blue`: interactive accent.
- `brand.sky`: soft selected/active background.
- `brand.ink`: high-priority text.
- `brand.muted`: secondary text.
- `brand.paper`: default app background.
- `brand.surface`: cards, panels, sheets.

## Semantic palette
- Success: green, quiet not neon.
- Warning: amber, used sparingly.
- Error: red, clear and accessible.
- Info: blue, used for hints and learning context.

## Rules
- Do not use more than one strong accent in one screen.
- Primary CTA must be visually obvious.
- Error color is only for actual errors, destructive actions, or validation.
- Disabled state uses opacity + cursor + text contrast, not color only.
