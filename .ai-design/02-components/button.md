# Button Component Spec

## Variants
- Primary: main action.
- Secondary: safe alternative.
- Ghost: low emphasis.
- Destructive: dangerous action.
- Link: text navigation.

## Size
- Mobile minimum touch target: 44px height.
- Desktop default: 40px height.
- Icon button: minimum 40px square.

## Rules
- One primary CTA per panel whenever possible.
- Loading button must disable duplicate submit.
- Button text must be explicit: avoid vague "OK" unless dialog context is obvious.
