# Screen — Realtime Battle

## Goal
Game-like but controlled.

## Layout
- Main question arena.
- Score/progress bar.
- Participant status compact.
- Chat and online list accessible simultaneously where screen allows.

## Mobile
- Online list collapses behind button with count badge.
- Chat remains accessible without blocking question.

## Rules
- Do not duplicate participant info.
- Bot tooltip/popover should not compete with main panel.
- Realtime status must be clear but not distracting.
