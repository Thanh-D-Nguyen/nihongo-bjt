# Screen — Search / Dictionary

## Goal
Fast lookup with premium learning context.

## Layout
Desktop:
- Left column result list.
- Right detail panel.
- Sticky search/filter bar.

Mobile:
- Search first.
- Detail opens as sheet/full page.
- Quick actions at bottom if needed.

## Must include
- Japanese headword
- reading
- meaning by language
- part of speech
- examples
- bookmark/add flashcard
- audio/TTS action if available
- skeleton loading
- no-result guidance

## Avoid
- emoji no-result.
- cramped kanji blocks.
- hiding core definition behind accordions.
