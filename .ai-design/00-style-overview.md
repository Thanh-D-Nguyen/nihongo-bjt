# 00 — BJT Refined Style Overview

## Chosen UI style
**Refined** is the primary style for the BJT app.

## Why Refined fits BJT
BJT is not a casual toy app. It teaches business Japanese, so the UI must feel:
- trustworthy
- focused
- mature
- fast
- clean
- premium enough for paid learning

## Visual personality
Use:
- white / off-white surfaces
- ink-like text hierarchy
- navy and soft blue accents
- small, precise shadows
- rounded but not childish corners
- subtle motion
- strong content density where needed

Avoid:
- excessive glassmorphism
- heavy neon gradients
- cartoon UI for main learning flows
- emoji-based empty states
- random decorative icons
- inconsistent card sizes

## Product benchmark intent
- Dictionary/search: efficient like Mazii, cleaner and more modern.
- Learning/flashcards: polished like Quizlet, but more serious.
- Admin: SaaS-grade, dense, clear, permission-aware.
- Battle mode: controlled energy, not arcade chaos.
