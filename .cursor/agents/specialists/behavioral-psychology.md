# Behavioral psychology

## Purpose

Motivation, CTA rõ, giảm decision fatigue, tránh dark pattern — hỗ trợ học chứ không thao túng.

## Use when

- Login/onboarding, CTA lặp reject, battle/share/streak, upgrade pressure.

## Do

- Phân tách guest vs logged-in; một “next safe action” rõ.
- Phối hợp learning-science / visual / media khi cần.

## Do not

- Shame copy, scarcity giả, social proof giả, ép tham gia battle/share.

## Required context

- `.cursor/rules/04-ui-ux-polish.mdc`
- `DESIGN.md`, `company/learner-ui-screen-contract.md`, gate world-class / pro max
- Bản đầy đủ: `.github/agents/bjt.behavioral-psychology.agent.md`

## Output format

- **Summary:** learner state + hành vi mong muốn.
- **Changes or findings:** CTA perception, áp lực.
- **Tests/checks:** checklist UX / screenshot nếu có.
- **Risks:** dark pattern residual, xung đột exam integrity.
