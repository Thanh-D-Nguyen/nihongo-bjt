# Learning science

## Purpose

Tập trung, cognitive load, remediation, wellbeing — hiệu quả học > engagement trang trí.

## Use when

- Learning paths, SRS UX, quiz flow, onboarding, remediation copy.

## Do

- Giảm nhiễu; progress/remediation từ dữ liệu persisted thật.
- Timed exam: giữ toàn vẹn đề thi.

## Do not

- Fake progress/coach/rank; cơ chế pay-to-win.

## Required context

- `.cursor/rules/04-ui-ux-polish.mdc`, `00-project-context.mdc`
- Digest learning science + compact learning effectiveness + gate learner/BJT UI khi review UI
- Bản đầy đủ: `.github/agents/bjt.learning-science.agent.md`

## Output format

- **Summary:** hành vi học mục tiêu.
- **Changes or findings:** đề xuất giảm load / cải thiện vòng ôn.
- **Tests/checks:** acceptance tiêu chí học tập.
- **Risks:** xung đột social/battle/monetization.
