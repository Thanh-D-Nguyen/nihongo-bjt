# TTS Reference Audio

Place reference audio files here for OmniVoice voice cloning.

## Requirements

- **Format**: WAV (16kHz or 24kHz, mono)
- **Duration**: 3–10 seconds (optimal for voice cloning)
- **Content**: Clear, clean speech — no background noise
- **Language**: Japanese (for standard pronunciation)

## Recommended Files

| File | Purpose | Description |
|------|---------|-------------|
| `ja-female-announcer.wav` | BJT Listening | NHK-style female announcer, formal business Japanese |
| `ja-male-announcer.wav` | BJT Listening (male) | NHK-style male announcer, formal business Japanese |
| `ja-casual-female.wav` | Battle Mode | Slightly more conversational tone |

## Where to Get Reference Audio

1. **NHK NEWS WEB EASY** — Download short clips from news readings
2. **JLPT listening samples** — Extract clean announcer segments
3. **Record your own** — Use a native speaker reading 3-5 formal sentences

## Tips

- Avoid reference clips with music, sound effects, or multiple speakers
- The cleaner the reference, the better the cloning quality
- For BJT, formal/keigo-style speech is preferred
- Cross-lingual accent: if ref is Japanese, all output will sound native Japanese
